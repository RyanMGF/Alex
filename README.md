# Landing Page - Curso de História

## Configuração da Imagem de Fundo

Para adicionar a foto da cidade como background no hero, siga os passos:

1. **Coloque a imagem na pasta do projeto** com o nome: `cidade-background.jpg`
   - A imagem deve estar no mesmo diretório que o arquivo `index.html`
   - Formato recomendado: JPG ou PNG
   - Dimensão recomendada: 1920x1080px ou maior para melhor qualidade

2. **A imagem será aplicada automaticamente** com:
   - Cobertura total do hero (cover)
   - Overlay escuro (70% de opacidade) para melhorar legibilidade do texto
   - Efeito parallax (fixed background attachment)

## Links Hotmart

Todos os botões de CTA estão configurados para levar a:
```
https://hotmart.com/seu-link-aqui
```

**Substitua "seu-link-aqui"** pelo link real da sua página Hotmart antes de publicar.

Para atualizar, procure por `https://hotmart.com/seu-link-aqui` no arquivo `index.html` e substitua por seu link real.

## Estrutura

- **index.html** - Página principal
- **styles.css** - Estilos (Dark Mode com verde limão #BFFF00)
- **script.js** - Contador regressivo
- **mockup-tablet.svg** - Imagem do mockup na seção hero
- **cidade-background.jpg** - *Adicione sua imagem aqui*
